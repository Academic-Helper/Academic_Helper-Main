# **App Name**: AcademiaConnect

## Core Features:

- User Authentication: User authentication and role management using Firebase Authentication, differentiating between assignment seekers, writers, and admins.
- Assignment Submission: Assignment submission form with fields for title, education level, subject, description (rich text), and file upload.
- Live Chat: Real-time chat functionality between assignment seekers and writers after an assignment is claimed, without storing history.
- Filtered Assignment Display: Display assignment postings tailored to the education level of logged in assignment writers.
- Admin Dashboard: Simple admin dashboard for user and assignment management.
- Assignment Claim/Release: Provide a 'Catch Assignment' and 'Give Up Assignment' function, updating the assignment status to claimed/open.

## Style Guidelines:

- Primary color: Light teal (#A7DBD8) for a calming and trustworthy feel, suitable for academic collaboration.
- Background color: Very light greenish-blue (#F0F8F7), nearly white, to keep the emphasis on the content.
- Accent color: Soft cyan (#91D8E4) for interactive elements and highlights to guide user attention.
- Body text: 'PT Sans' (sans-serif) for easy readability. Headline font: 'Playfair' (serif) for elegance in titles, but avoiding for the body of longer texts.
- Use clear, professional icons to represent actions and assignment statuses.
- Maintain a clean, minimal layout with clear separation of content sections. Prioritize role-based navigation with easily identifiable links (e.g., 'Post Assignment', 'Catch Assignment', 'Admin Login').
- Use subtle transitions to enhance user experience without being distracting.
